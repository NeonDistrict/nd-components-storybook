# Toast

## Setup

This component needs a provider to be used.

```tsx  title="/src/App.tsx"
import { ToastContainer } from '@blockadegames/pizza-juice';

export const App = () => {
  return (
    <SomeProviders>
      {/* a lot of providers/routing stuff */}
      <ToastContainer />
    </SomeProviders>
  );
};
```

## Import

```jsx
import { ToastContainer } from '@blockadegames/pizza-juice';
```
