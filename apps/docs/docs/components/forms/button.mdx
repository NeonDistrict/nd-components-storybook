# Button

Button component is used to trigger an action or event, such as submitting a form, opening a Modal, canceling an action, or performing a delete operation.

## Import

```jsx
import { Button } from '@blockadegames/pizza-juice';
```

## Usage

<Button>Neon District</Button>

---

```jsx
<Button>Neon District</Button>
```

### Props

| Attribute    | Type                      | Description                   | Default   |
| ------------ | ------------------------- | ----------------------------- | --------- |
| color        | [Color](#color)           | Change button color           | `default` |
| variant      | [Variant](#variant)       | Change button style           | `false`   |
| disabled     | `boolean`                 | Disable button                | `false`   |
| shape        | [Shape](#shape)           | Change shape of button        | `squared` |
| fluid        | `boolean`                 | Full width button             | `false`   |
| loading      | `boolean`                 | Show loading spinner          | `false`   |
| icon         | `JSX.Element`             | Show icon in button           | -         |
| iconPosition | `left` `right`            | Select position to show icon  | `left`    |
| type         | `button` `reset` `submit` | Native type of button element | `button`  |
| ...          | `'id', 'className', ...`  | Button native props           | -         |
