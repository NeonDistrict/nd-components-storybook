# Button

Button component is used to trigger an action or event, such as submitting a form, opening a Modal, canceling an action, or performing a delete operation.


## Import
```jsx
import { Button } from '@blockadegames/pizza-juice'
```

## Usage

<Button>Neon District</Button>

- - -

```jsx
<Button>Neon District</Button>
```

## Color

Variant `color` of the button.

<Stack wrap='wrap'>
  <Button color="primary">Primary</Button>
  <Button color="secondary">Secondary</Button>
  <Button color="destructive">Destructive</Button>
  <Button color="warning">Warning</Button>
  <Button color="success">Success</Button>
</Stack>

## Variant

Types of `variant` available:

<Stack wrap='wrap'>
  <Button variant="solid">Solid</Button>
  <Button variant="outlined">Outlined</Button>
  <Button variant="naked">Naked</Button>
  <Button variant="unstyled">Unstyled</Button>
</Stack>
