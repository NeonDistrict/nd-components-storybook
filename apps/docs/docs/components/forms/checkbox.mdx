# Checkbox

Checkbox component is used in forms when a user needs to select multiple values from several options.

## Import
```jsx
import { Checkbox } from '@blockadegames/pizza-juice'
```

## Usage

<Stack>
  <Checkbox size="sm"> Checkbox sm </Checkbox>
  <Checkbox size="md"> Checkbox md </Checkbox>
  <Checkbox size="lg"> Checkbox lg </Checkbox>
</Stack>

- - -

```jsx
<Checkbox> Checkbox </Checkbox>
```
