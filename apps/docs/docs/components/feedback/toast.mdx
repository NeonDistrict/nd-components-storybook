# Toast

## Setup

This component needs a provider to be used.

```tsx title="/src/App.tsx"
import { ToastContainer, toast } from '@blockadegames/pizza-juice';

export const App = () => {
  return (
    <SomeProviders>
      {/* a lot of providers/routing stuff */}
      <ToastContainer />
    </SomeProviders>
  );
};
```

## Import

```jsx
import { ToastContainer, toast } from '@blockadegames/pizza-juice';
```

## Usage

<div>
  <Button
    onClick={() =>
      toast({
        variant: 'icon',
        type: 'success',
        title: 'Success Title',
        message: 'Lorem ipsum success',
      },
      {
       position: 'bottom-center',
      })
    }
  >
    Show toast
  </Button>

  <ToastContainer />
</div>

---

```jsx
toast(
  {
    variant: 'icon', // 'image' | 'timer'
    type: 'success', // 'warning' | 'error' | 'default'
    title: 'Success Title',
    message: 'Lorem ipsum success',
    // if only a line of message isn't enought, you can pass subitems
    // subitems: <>{aLotOfElements()}</>
    // button1: <Button>Primary Action</Button>
    // button2: <Button>Secondary Action</Button>
  },
  {
    // the second object is opcional. You can pass toast options, like:
    position: 'bottom-center',
  },
);
```
