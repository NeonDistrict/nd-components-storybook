# Progress

Progress is used to display the progress status for a task that takes a long time or consists of several steps.

## Import

```jsx
import { Progress } from '@blockadegames/pizza-juice'
```

## Usage

<Progress  
  value={40}
  size="md"
  min={0}
  max={100}
  color="red"
/>
