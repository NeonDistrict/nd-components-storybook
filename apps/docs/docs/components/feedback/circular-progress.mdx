# Circular Progress

Circular Progress is used to indicates the progress for determinate and indeterminate processes.

## Import

```jsx
import { CircularProgress } from '@blockadegames/pizza-juice'
```

## Usage

<CircularProgress  
  value={75}
  size="md"
  thickness="lg"
  min={0}
  max={100}
  color="green"
/>
