# Spinner

Spinner is used to display the Spinner status for a task that takes a long time or consists of several steps.

## Import

```jsx
import { Spinner } from '@blockadegames/pizza-juice';
```

## Usage

<Spinner
  css={{
    $$color: '$colors$teal-500',
    $$size: '50px',
    $$thickness: 2,
    $$speed: '0.6s',
  }}
/>

---

```jsx
<Spinner
  css={{
    $$color: '$colors$teal-500',
    $$size: '50px',
    $$thickness: 2,
    $$speed: '0.6s',
  }}
/>
```

### CSS Variables

| Attribute   | Type     | Description            | Default        |
| ----------- | -------- | ---------------------- | -------------- |
| $$color     | `string` | Color                  | `currentColor` |
| $$size      | `string` | Size                   | `16px`         |
| $$thickness | `number` | Thickness              | `1.5`          |
| $$speed     | `string` | Speed of the animation | `0.4s`         |
