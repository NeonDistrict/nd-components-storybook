import { Alert, Button } from '@blockadegames/pizza-juice'

# Alert

Alerts are used to communicate a state that affects a system, feature or page.

## Import

```jsx
import { Alert } from '@blockadegames/pizza-juice'
```

## Usage

<Alert   
  title="There was an error"
  variant="primary"
  subtitle="Error message"
  description="Lorem ipsum dolor sit amet, consectetur adipiscing elit"
  dismissible={false}
>
  <Button>
    OK, I got it
  </Button>
</Alert>

---

```jsx
<Alert
  title="There was an error"
  variant="Error something"
  subtitle="Subtitle"
  description="Description"
>
  <Button>
    OK, I got it
  </Button>
</Alert>
```
