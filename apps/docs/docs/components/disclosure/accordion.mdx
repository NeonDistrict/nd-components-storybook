# Accordion

Accordions display a list of high-level options that can expand/collapse to reveal more information.

## Import

```jsx
import { Accordion, AccordionItem } from '@blockadegames/pizza-juice';
```

## Usage

<Accordion defaultValue="1">
  <AccordionItem title="Username" value="1">
    Username content
  </AccordionItem>

  <AccordionItem title="Email" value="2">
    Email content
  </AccordionItem>
</Accordion>
