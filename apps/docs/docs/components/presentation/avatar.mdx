# Avatar

The Avatar component is used to represent user, and displays the profile picture, initials or fallback icon.

## Import

```jsx
import { Avatar } from '@blockadegames/pizza-juice'
```

## Usage

<Avatar 
  size="160"
  src="https://bit.ly/ben-heidorn"
  alt="Ben Heidorn"
  objectFit={true}
  cover={true}
/>
