# Tooltip

A tooltip is a brief, informative message that appears when a user interacts with an element.

## Import

```jsx
import { Tooltip } from '@blockadegames/pizza-juice';
```

## Usage

<Tooltip text="What is this?" position="right">
  <Button color="destructive">Hover me</Button>
</Tooltip>

---

```jsx
<Tooltip text="What is this?" position="right">
  {/* children */}
</Tooltip>
```
