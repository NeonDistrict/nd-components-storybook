# Tooltip

A tooltip is a brief, informative message that appears when a user interacts with an element.

This component need a provider to work, you can use the `TooltipProvider` component to wrap your app.

## Import

```jsx
import { Tooltip } from '@blockadegames/pizza-juice';
```

## Usage

<Tooltip text="What is this?" position="right">
  <Button color="destructive">Hover me</Button>
</Tooltip>

### Props

| Attribute     | Type                          | Description                                                                                                      | Default |
| ------------- | ----------------------------- | ---------------------------------------------------------------------------------------------------------------- | ------- |
| open          | `boolean`                     | The controlled open state of the tooltip. Must be used in conjunction with `onOpenChange`.                       | `-`     |
| defaultOpen   | `boolean`                     | The open state of the tooltip when it is initially rendered. Use when you do not need to control its open state. | `-`     |
| delayDuration | `number`                      | The duration from when the mouse enters a tooltip trigger until the tooltip opens.                               | `700`   |
| text          | `string`                      | Content of Tooltip                                                                                               | `-`     |
| position      | `top` `right` `bottom` `left` | The preferred position of the trigger to render against when open                                                | `-`     |
| sideOffset    | `number`                      | The distance in pixels from the trigger.                                                                         | `5`     |
| onOpenChange  | `(open: boolean) => void`     | Event handler called when the open state of the tooltip changes.                                                 | `-`     |
