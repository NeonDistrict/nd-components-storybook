# Box

Acts like a div element, it can be used to group other elements. 

It also gives access to the powerful `css` and `as` property, these are more explained in the [Advanced Usage](later) section.

## Import

```jsx
import { Box } from '@blockadegames/pizza-juice'
```

## Usage

<Box css={{ p: '$4', w: 'max-content', bg: '$red-550', color: "$white" }}>
  Hello, World!
</Box>
