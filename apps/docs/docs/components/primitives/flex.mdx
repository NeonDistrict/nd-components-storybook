# Flex

It is a flex container that can be used to layout elements in a `row` or `column`.

It also gives access to the powerful `css` and `as` property, these are more explained in the [How?] section.

## Import

```jsx
import { Flex } from '@blockadegames/pizza-juice'
```

## Usage

`Flex` has the most used flexbox properties as props.

### Direction

Acts like CSS `flex-direction` property.

<Flex direction="column-reverse" gap={4}>
  <Center css={{ size: 30, br: '$sm', bg: '$grey-600' }}>1</Center>
  <Center css={{ size: 30, br: '$sm', bg: '$grey-600' }}>2</Center>
  <Center css={{ size: 30, br: '$sm', bg: '$grey-600' }}>3</Center>
</Flex>

- - -
```jsx
<Flex direction="column-reverse" gap={4}>
  <Center css={{ size: 30, br: '$sm', bg: '$grey-600' }}>1</Center>
  <Center css={{ size: 30, br: '$sm', bg: '$grey-600' }}>2</Center>
  <Center css={{ size: 30, br: '$sm', bg: '$grey-600' }}>3</Center>
</Flex>
```


### Justify

Acts like CSS `justify-content`, it aligns in the main axis

<Flex justify="center" css={{ bg: '$grey-700', br: '$sm', p: '$4' }}>
  <Text weight="medium">Look mama, I'm centered!</Text>
</Flex>

- - - 

```jsx
<Flex justify="center">
  Look mama, I'm centered!
</Flex>
```
### Align

Acts like CSS `align-items`, it aligns in the cross axis

<Flex align="end" css={{ bg: '$grey-700', br: '$sm', p: '$4', h: 100 }}>
  <Text weight="medium">In the eeeeeeenddddd, it doesn't even matter</Text>
</Flex>

- - - 

```jsx
<Flex align="end" css={{ bg: '$grey-700', br: '$sm', p: '$4', h: 100 }}>
  <Text weight="medium">In the eeeeeeenddddd, it doesn't even matter</Text>
</Flex>
```

### Gap

Acts like CSS `gap`, it creates a gap between the elements

<Flex gap={8} css={{ bg: '$grey-700', br: '$sm', p: '$4' }}>
  <Box css={{ size: 30, br: '$sm', bg: '$pink-500' }}/>
  <Box css={{ size: 30, br: '$sm', bg: '$green-500' }}/>
  <Box css={{ size: 30, br: '$sm', bg: '$teal-500' }}/>
</Flex>

- - - 

```jsx
<Flex gap={8} css={{ bg: '$grey-700', br: '$sm', p: '$4' }}>
  <Box css={{ size: 30, br: '$sm', bg: '$pink-500' }}/>
  <Box css={{ size: 30, br: '$sm', bg: '$green-500' }}/>
  <Box css={{ size: 30, br: '$sm', bg: '$teal-500' }}/>
</Flex>
```
