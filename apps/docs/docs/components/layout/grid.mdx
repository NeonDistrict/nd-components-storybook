# Grid

Grid is a layout for managing grid layouts.

## Import

```jsx
import { Grid } from '@blockadegames/pizza-juice';
```

#### Shorthand Props

Using the `Grid` component, here are some helpful shorthand(maybe) props:

- `gap` is `gap`
- `flow` is `gridAutoFlow`
- `columns` is `gridTemplateColumns`
- `gapX` is `rowGap`
- `gapY` is `columnGap`
- `direction` is `flexDirection`
- `justify` is `justifyContent`
- `align` is `alignItems`
- `wrap` is `flexWrap`
