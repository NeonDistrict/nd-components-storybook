# Grid System

Powerful flexbox grid to build layouts of all shapes and sizes thanks to a twelve column system.

## Import

```jsx
import { Row, Col } from '@blockadegames/pizza-juice';
```

## Usage

<Row>
  <Col
    col={{ '@initial': 12, '@sm': 2, '@md': 4 }}
    css={{ bg: '$pink-500', h: 50 }}
  />
  <Col
    col={{ '@initial': 12, '@sm': 2, '@md': 4 }}
    css={{ bg: '$yellow-500', h: 50 }}
  />
  <Col
    col={{ '@initial': 12, '@sm': 2, '@md': 4 }}
    css={{ bg: '$red-500', h: 50 }}
  />
  <Col
    col={{ '@initial': 12, '@sm': 2, '@md': 4 }}
    css={{ bg: '$teal-500', h: 50 }}
  />
  <Col
    col={{ '@initial': 12, '@sm': 2, '@md': 4 }}
    css={{ bg: '$grey-700', h: 50 }}
  />
  <Col
    col={{ '@initial': 12, '@sm': 2, '@md': 4 }}
    css={{ bg: '$green-500', h: 50 }}
  />
</Row>
