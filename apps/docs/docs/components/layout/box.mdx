# Box

`Box` is the most abstract component. By default, it renders a `div` element

## Import

```jsx
import { Box } from '@blockadegames/pizza-juice'
```

## Usage

<Box css={{ p: '$4', w: 'max-content', bg: '$red-550', color: "$white" }}>
  Hello, World!
</Box>
