# Center

Center is a layout component that centers its child within itself.

## Import

```jsx
import { Center } from '@blockadegames/pizza-juice';
```

#### Shorthand Props

Using the `Center` component, here are some helpful shorthand(maybe) props:

- `gap` is `gap`
- `direction` is `flexDirection`
- `justify` is `justifyContent`
- `align` is `alignItems`
- `wrap` is `flexWrap`

## Usage

<Center css={{ h: 100, bg: '$red-550' }}>This is the Center</Center>
