# Badge

Badges are used to highlight an item's status for quick recognition.

## Import
```jsx
import { Badge } from '@blockadegames/pizza-juice'
```

## Usage

<Stack wrap='wrap'>
  <Badge rarity='common' />
  <Badge rarity='uncommon' />
  <Badge rarity='rare' />
  <Badge rarity='ultraRare' />
  <Badge rarity='legendary' />
</Stack>

- - -

```jsx
<Badge rarity="rare" />
```
