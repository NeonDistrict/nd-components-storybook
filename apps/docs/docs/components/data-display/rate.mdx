# Rate

A rating indicates user interest in content.

## Import

```jsx
import { Rate } from '@blockadegames/pizza-juice';
```

## Usage

<Rate value={3} />

---

```jsx
<Rate value={3} />
```

### Props

| Attribute | Type                | Description             | Default |
| --------- | ------------------- | ----------------------- | ------- |
| value     | `1` `2` `3` `4` `5` | Number of stars checked | -       |
