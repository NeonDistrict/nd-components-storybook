# Image

The Image component is used to display images with support for fallback.

## Import

```jsx
import { Image } from '@blockadegames/pizza-juice';
```

## Usage

<Image
  src="https://bit.ly/ben-heidorn"
  alt="Benjamin Heidorn"
  css={{ size: 200 }}
/>

---

```jsx
<Image src="https://bit.ly/ben-heidorn" alt="Benjamin Heidorn" />
```
