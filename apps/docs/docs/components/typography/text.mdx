# Text

Text is the used to render text and paragraphs within an interface.

## Import

```jsx
import { Text } from '@blockadegames/pizza-juice';
```

## Usage

It uses the [typography](/docs/tokens/typography) tokens.

<Text weight="medium" size="xl" css={{ color: '$pink-500' }}>
  Neon Pizza
</Text>

---

```jsx
<Text weight="medium" size="xl" css={{ color: '$pink-500' }}>
  Neon Pizza
</Text>
```

### Left line

You can add a left line to the text. This is useful for titles.

<Text
  as="h1"
  transform="uppercase"
  weight="medium"
  size="2xl"
  leftLine
  css={{
    $$lineColor: '$colors$red-500',
    $$lineSpacing: '$space$4',
  }}
>
  Neon District
</Text>

---


### Props

| Attribute | Type                                  | Description           | Default |
| --------- | ------------------------------------- | --------------------- | ------- |
| size      | [Typography](/docs/tokens/typography) | font-size             | `-`     |
| weight    | [Typography](/docs/tokens/typography) | font-weight           | `-`     |
| textAlign | `left` `center` `right`               | text-align            | `-`     |
| transform | `uppercase` `lowercase` `normal`      | text-transform        | `-`     |
| leftLine  | `boolean`                             | Add left line of text | `-`     |

### CSS Variables

| Attribute     | Type     | Description                 | Default         |
| ------------- | -------- | --------------------------- | --------------- |
| $$lineColor   | `string` | Line color                  | `$colors$white` |
| $$lineSpacing | `string` | Space between text and line | `$space$4`      |
