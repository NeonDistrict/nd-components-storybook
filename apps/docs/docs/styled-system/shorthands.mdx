# Shorthands

Shorthand to work with css properties.

## Usage

```js
m: 'margin',
mt: 'marginTop',
mr: 'marginRight',
mb: 'marginBottom',
ml: 'marginLeft',
mx: ['marginLeft', 'marginRight'],
my: ['marginTop', 'marginBottom'],
p: 'padding',
pt: 'paddingTop',
pr: 'paddingRight',
pb: 'paddingBottom',
pl: 'paddingLeft',
px: ['paddingLeft', 'paddingRight'],
py: ['paddingTop', 'paddingBottom'],
bg: 'background',
size: ['width', 'height'],
maxSize: ['maxWidth', 'maxHeight'],
minSize: ['minWidth', 'minHeight'],
br: 'borderRadius',
bs: 'boxShadow',
justify:'justifyContent',
align:'alignItems',
w: 'width',
maxW: 'maxWidth',
minW: 'minWidth',
h: 'height',
maxH: 'maxHeight',
minH: 'minHeight',
d: 'display',
columns: 'gridTemplateColumns',
rows: 'gridTemplateRows',
blur: `blur(${value})`,
textGradient:['backgroundImage','WebkitBackgroundClip'.'WebkitTextFillColor']
```
