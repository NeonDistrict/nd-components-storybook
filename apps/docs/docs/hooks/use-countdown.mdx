# useCountdown

`useCountdown` is a custom hook used to help manage the state of a countdown timer.

## Import

```jsx
import { useCountdown } from '@blockadegames/pizza-juice';
```

## Arguments

The `useCountdown` hook takes the following arguments:

| Name      | Type     | Required | Description                 |
| --------- | -------- | -------- | --------------------------- |
| `endDate` | `number` | `true`   | End date in unix timestamp. |

## Return value

The `useCountdown` hook returns an object with the following fields:

| Name               | Type      | Default | Description                      |
| ------------------ | --------- | ------- | -------------------------------- |
| `days`             | `string`  |         | Formatted days with padLeft.     |
| `hours`            | `string`  |         | Formatted hours with padLeft.    |
| `minutes`          | `string`  |         | Formatted minutes with padLeft.  |
| `seconds`          | `string`  |         | Formatted seconds with padLeft.  |
| `daysAsNumber`     | `number`  |         | Days as raw format.              |
| `hoursAsNumber`    | `number`  |         | Hours as raw format .            |
| `minutesAsNumber`  | `number`  |         | Minutes as raw format.           |
| `secondsAsNumber`  | `number`  |         | Seconds as raw format.           |
| `secondsRemaining` | `number`  |         | Seconds remaining as raw format. |
| `isTimerDone`      | `boolean` |         | Check if countdown ends.         |

## Usage

export const Example = () => {
  const endDate = new Date(new Date().getTime() + 86400000 * 2).getTime(); // 2 days
  const { days, hours, minutes, seconds } = useCountdown(endDate);
  return (
    <>
      <Text size="4xl" weight="bold">
        {days} : {hours} : {minutes} : {seconds}
      </Text>
    </>
  );
};

```jsx
import { useCountdown } from '@blockadegames/pizza-juice';

() => {
  const endDate = new Date(new Date().getTime() + 100000).getTime(); // 100 seconds
  const { days, hours, minutes, seconds } = useCountdown(endDate);

  return (
    <Text size="4xl" weight="bold">
      {days} : {hours} : {minutes} : {seconds}
    </Text>
  );
};
```
