# useId

This hook generates unique IDs i.e.

## Import

```jsx
import { useId } from '@blockadegames/pizza-juice';
```

## Arguments

The `useId` hook takes the following arguments:

| Name       | Type     | Required | Description                                |
| ---------- | -------- | -------- | ------------------------------------------ |
| `initials` | `string` | `false`  | Return random Id with `initials_` + value. |

## Return value

The `useId` hook returns an object with the following fields:

| Name    | Type     | Default | Description |
| ------- | -------- | ------- | ----------- |
| `value` | `string` |         | Random id.  |

## Usage

```jsx
import { useId } from '@blockadegames/pizza-juice';

() => {
  const id = useId('button');

  return (
    // ...
  )
};
```
