# Responsive Styles

How to configure breakpoints and apply variants responsively.

## Breakpoints

```js
sm: '576px',
md: '768px',
lg: '992px',
xl: '1200px',
'2xl': '1400px',
```

## Usage

You must use the `@initial` breakpoint to declare the initial variant before any breakpoints are applied.

Once configured, you can apply different variants at different breakpoints.

In the example below, we apply the violet color variant at the bp2 condition.

```jsx
import { styled } from '@blockadegames/pizza-juice';

const Button = styled('button', {
  // base styles

  variants: {
    color: {
      success: {},
      error: {},
    },
  },
});

() => (
  <Button
    color={{
      '@initial': 'success',
      '@lg': 'error',
    }}
  />
);
```
